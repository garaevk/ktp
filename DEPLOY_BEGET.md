# Деплой КТП-генератора на Beget

## 1. Регистрация
1. Зайди на https://beget.com
2. Нажми «Регистрация»
3. Заполни ФИО, email, телефон
4. Подтверди email

## 2. Создание Python-приложения
1. Войди в панель управления: https://cp.beget.com
2. Перейди в раздел **«Python»** (в меню слева)
3. Нажми **«Создать приложение»**
4. Заполни:
   - **Домен**: выбери бесплатный поддомен `*.beget.tech` или свой домен
   - **Корневая папка**: `ktp_app` (или создай папку и укажи её)
   - **Версия Python**: 3.12 (или доступная последняя)
   - **Точка входа**: `passenger_wsgi:application`
5. Нажми **«Создать»**

## 3. Загрузка файлов
1. В панели управления перейди в **«Файловый менеджер»**
2. Найди папку своего домена (обычно `~/domains/твой-домен.beget.tech/`)
3. Загрузи туда **все файлы** из папки `ktp_app`:
   - `app.py`
   - `ktp_logic.py`
   - `passenger_wsgi.py`
   - `requirements.txt`
   - Папка `static/`
   - Папка `templates/`
4. **Не загружай** папку `sessions/` и `__pycache__/`

## 4. Установка зависимостей
1. В панели управления перейди в раздел **«Python»**
2. Найди своё приложение, нажми **«Управление»**
3. Перейди на вкладку **«Модули»** или **«Зависимости»**
4. Нажми **«Установить из requirements.txt»**
5. Дождись завершения установки

## 5. Настройка прав доступа
1. В файловом менеджере убедись, что папка `sessions` существует
2. Если её нет — создай папку `sessions` в корне приложения
3. Установи права **755** на папку `sessions`

## 6. Проверка
1. Открой `http://твой-домен.beget.tech`
2. Если сайт не загружается — проверь логи в разделе Python → «Логи»

## Важно
- Файлы `.docx` загружаются во временную папку, которая очищается автоматически
- Сессии хранятся в папке `sessions/` — она должна быть доступна для записи
- Максимальный размер загружаемого файла: 32 МБ (настроено в `app.py`)
